# Οδηγός Ενημέρωσης Thronos Chain (v2.1)

Ακολουθήστε τα παρακάτω βήματα για να ενημερώσετε τον server σας με ασφάλεια.

## 1. Αντίγραφο Ασφαλείας (Backup)
**ΠΡΟΣΟΧΗ:** Πριν κάνετε οποιαδήποτε ενέργεια, κρατήστε ένα πλήρες αντίγραφο (backup) του τρέχοντος φακέλου του server σας. Αυτό διασφαλίζει ότι μπορείτε να επαναφέρετε το σύστημα αν κάτι πάει στραβά.

## 2. Εγκατάσταση
Αντιγράψτε όλα τα περιεχόμενα αυτού του πακέτου (φάκελος `thronosChain-main`) μέσα στον φάκελο του live server σας.
- Όταν ερωτηθείτε, επιλέξτε **"Αντικατάσταση όλων" (Overwrite all)** για τα αρχεία που υπάρχουν ήδη.

## 3. Ασφάλεια Δεδομένων
Μην ανησυχείτε για τα δεδομένα σας.
- Τα αρχεία `blockchain.json`, `wallets.json`, `ledger.json` και άλλα αρχεία βάσης δεδομένων έχουν **αφαιρεθεί σκόπιμα** από αυτό το πακέτο ενημέρωσης.
- Αυτό σημαίνει ότι η αντιγραφή των αρχείων **ΔΕΝ** θα διαγράψει το ιστορικό των συναλλαγών ή τα πορτοφόλια των χρηστών σας.

## 4. Επανεκκίνηση
Μετά την αντιγραφή των αρχείων, πρέπει να επανεκκινήσετε τον server για να εφαρμοστούν οι αλλαγές.
Τρέξτε την εντολή:
```bash
python server.py
```

## 5. Νέες Λειτουργίες
Μετά την επανεκκίνηση, οι παρακάτω νέες λειτουργίες θα είναι διαθέσιμες:
- **Addons**: Έχει δημιουργηθεί ένας νέος φάκελος `addons/`.
  - `addons/crypto_hunters`: Περιέχει το πλήρες πακέτο του παιχνιδιού (Backend + Mobile Demo).
  - `addons/ai_agent`: Περιέχει τον αυτόνομο AI Agent.
- **Recovery**: Το σύστημα ανάκτησης έχει αναβαθμιστεί με κωδικό πρόσβασης.
- **Mining**: Η λογική των αμοιβών έχει ενημερωθεί.

Καλή επιτυχία!
