# Items Shop

The Crypto Hunters shop sells items that help players complete missions, gain
more DRX or customise their avatars.  All purchases are paid with game DRX
credits, which are off‑chain but redeemable for on‑chain DRX via the withdraw
queue.

| Item             | ID           | Description                                      | Price (DRX) |
|------------------|-------------|--------------------------------------------------|-------------|
| Security Booster | `I_SEC_BOOST`| Temporary resistance to threats during missions  | 150         |
| Hash Toolkit     | `I_HASH_TOOL`| Solves algorithmic puzzles faster                | 200         |
| DRX Doubler (30m)| `I_DBL_REWARD`| Doubles mission rewards for 30 minutes          | 400         |
| NFT Shard        | `I_NFT_SHARD`| Combine shards to mint a rare NFT                | 800         |
| Hunter Skin: Neon| `I_COS_SKIN1`| Cosmetic outfit (no gameplay effect)             | 120         |
