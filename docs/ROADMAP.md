# Thronos Chain Product Roadmap

## Core Services Milestones

### Learn2Earn (L2E) — Core Thronos Chain Service

**Roadmap wording (added):**
> Learn2Earn is now recognized as a core Thronos Chain service with phased delivery history:
> - Phase 1: core foundation
> - Phase 2: academic operations
> - Phase 3: approval and issuance
> - Phase 4: RBAC, tenant enforcement, audit trail

#### Delivery Summary
- **Phase 1 (core foundation):** THR-first course pricing, on-chain aligned enrollment flows, controlled reward model baseline.
- **Phase 2 (academic operations):** weighted quizzes, pass/fail separation, certificate eligibility model, teacher-facing academic controls.
- **Phase 3 (approval and issuance):** explicit non-automatic certificate approval and issuance operations, queue-oriented workflows.
- **Phase 4 (RBAC/tenant/audit):** role boundaries, tenant enforcement hooks, auditable approval/issuance history endpoints.

#### Current Status
- Learn2Earn is production-grade as a core service.
- Governance/dashboard depth and compliance exports are still in refinement scope (not represented as fully complete).
