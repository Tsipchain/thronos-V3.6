# Thronos Network Logos

This directory contains the official Thronos Network branding assets.

## Logo Files

Please add the following logo files to this directory:

### THR Token Logos
- `thr-coin-dark.png` - THR token with infinity chain on dark background (1024x1024)
- `thr-coin-transparent.png` - THR token with transparency (1024x1024)

### Thronos Chain Logos
- `thronos-chain-dark.png` - Thronos Chain 2025 with Bitcoin throne on dark background (1024x1024)
- `thronos-chain-transparent.png` - Thronos Chain 2025 with transparency (1024x1024)

### Icon Sizes (for Chrome Extension and Mobile)
- `icon-16.png` - 16x16 px
- `icon-32.png` - 32x32 px

- `icon-128.png` - 128x128 px

## Design Guidelines

### Colors
- Primary Green: `#00ff00`
- Secondary Green: `#00ff66`
- Gold/Yellow: `#ffd700` to `#ffed4e`
- Dark Background: `#000000` to `#0a0a0a`
- Circuit Board Green: `#1a3b2e`

### Typography
- Primary Font: "Courier New", monospace
- Accent Text: Bold, uppercase for THR/Thronos
- Tagline: "Pledge to the unburnable" and "Strength in every link, light in every Block"

## Usage

These logos are used across:
- Web application (base.html, wallet pages)
- Chrome Extension
- Mobile SDK (iOS, Android, React Native)
- Documentation and marketing materials
