Configuration Guide

Setting up API Keys:

To enable AI providers, set the following environment variables on your server:

  • OPENAI_API_KEY - OpenAI GPT models (gpt-4.1, gpt-4.1-mini, o3-mini)
  • ANTHROPIC_API_KEY - Anthropic Claude models (claude-3.5-sonnet, claude-3.5-haiku)
  • GEMINI_API_KEY or GOOGLE_API_KEY - Google Gemini models

Railway Deployment:

Add these keys in your Railway project's Variables tab. The server will automatically detect and enable the corresponding models.

Auto Mode:

When multiple providers are configured, the "Auto" option will intelligently route requests to the best available model based on task type and provider health.
