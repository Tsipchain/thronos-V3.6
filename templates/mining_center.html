🔌 Block Erupter & USB ASICs

Connect your USB ASIC (Block Erupter, Red Fury, GekkoScience, Antminer U1/U3) via cgminer with your V1 Thronos address as the username.

Step 1 — Start stratum_proxy.py

python stratum_proxy.py
# Proxy listens on localhost:3334
# Connects to thrchain.up.railway.app

Step 2 — Edit cgminer.conf

url: stratum+tcp://127.0.0.1:3334
user: YOUR_V1_ADDRESS
pass: x
icarus-options: 115200:1:1
hotplug: 5

Step 3 — Run cgminer

cgminer.exe -c cgminer.conf
# cgminer auto-detects the USB ASIC via hotplug
# Rewards go to your V1 THR address

💻 HTTP Mining API (CPU/GPU)

CPU and GPU miners can poll the work API directly without running the Stratum proxy.

GET Work: /api/miner/work?address=THR_ADDR
POST Submit: /api/miner/submit
Algorithm: SHA-256d (double SHA-256)

📦 Work Response Fields

targethex — mining target (find hash below this)
prev_hashhex — previous block hash
heightint — block height to mine
rewardfloat — THR reward for solving
nbitshex — compact difficulty bits
job_idstr — unique job identifier

📤 Submit Payload

{
  "thr_address": "YOUR_THR_ADDRESS",
  "nonce": 12345678,
  "job_id": "job_id_from_work",
  "merkle_root": "hex_string",
  "prev_hash": "hex_string",
  "time": 1718000000,
  "nbits": 486604799,
  "pow_hash": "hex_string"
}
